# Logo Assets

Place the company logo files in this directory.

## Recommended Files:
- `logo.png` or `logo.svg` - Main logo (preferred format: SVG for scalability)
- `logo-icon.png` - Icon version for favicon/small sizes
- `logo-white.png` - White version for dark backgrounds (if needed)

## Logo Specifications:
Based on the ConstructionGuru logo:
- Main logo should include the stylized graphic elements
- Company name: "ConstructionGuru"
- Tagline: "Real-Time Pricing for Construction Professionals"
- Colors: Blue (#0066CC or similar) and Orange (#FF6600 or similar)

## Usage:
The logo will be automatically referenced from `/images/logo.png` or `/images/logo.svg` in components.
